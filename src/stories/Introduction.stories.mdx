# Welcome to Storybook by DataSaysSo

**DataSaysSo** represents an effort to bring current affairs, historical data into nicely visualized format.
Some of the work which involves big data sets has been carried out on [Kaggle](https://www.kaggle.com/datasaysso "Kaggle"). 
The work which involves relatively small size of data has been visualized in this storybook.
